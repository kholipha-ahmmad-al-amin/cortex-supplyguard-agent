# Cortex Deep Reasoner Skill (`cortex-reasoner`)

## Overview
The **Cortex Deep Reasoner** skill uses Snowflake Cortex LLM capabilities (`SNOWFLAKE.CORTEX.COMPLETE` / Cortex AI functions) to bridge structured metrics and unstructured operational data (e.g. logistics incident reports, weather advisories, vendor emails, port congestion notices).
