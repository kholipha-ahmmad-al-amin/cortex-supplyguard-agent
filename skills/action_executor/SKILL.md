# Contextual Action Executor Skill (`action-executor`)

## Overview
The **Contextual Action Executor** skill converts LLM mitigation recommendations into transactional, auditable enterprise actions. It enforces strict organizational guardrails (e.g. expenditure limits, approval delegation) and executes database mutations or API hooks.
